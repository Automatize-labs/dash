# Directive: Analyze Data

## Goal
Analyze a CSV file and output summary statistics.

## Inputs
- `input_file`: Path to the CSV file to analyze.

## Outputs
- `summary.txt`: A text file containing summary statistics.

## Tools
- `execution/analyze_csv.py`

## Instructions
1.  Read the `input_file`.
2.  Calculate summary statistics (mean, median, mode) for numeric columns.
3.  Write the summary to `summary.txt`.
